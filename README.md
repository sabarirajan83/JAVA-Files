## 🚀 How to Run

1. Clone this repository:
    ```bash
    git clone https://github.com/your-username/java-code-repo.git
    ```
2. Navigate to the project directory:
    ```bash
    cd java-code-repo/src
    ```
3. Compile a Java file:
    ```bash
    javac Main.java
    ```
4. Run the program:
    ```bash
    java Main
    ```
